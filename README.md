## 換算種類

參考：[維基百科「台制」](https://zh.wikipedia.org/wiki/台制)。

* 重量：公克、公斤、台斤、台兩、英磅、盎司。
* 長度：公尺、公分、台尺、台寸、英尺、英寸。
* 面積：台坪、台畝、台分、台甲、m²（平方公尺）、公頃、ft²（平方英尺）。
* 貨幣：台幣、美元、日圓、歐元、英鎊、越南盾、港幣、韓元、人民幣。
